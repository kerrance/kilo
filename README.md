# Kilo

A body composition dashboard that visualises data exported from your smart scale app.

## How to use

1. Export your data from the **INSMART Health** app ([iOS](https://apps.apple.com/gb/app/insmart-health/id1633412395) / [Android](https://play.google.com/store/apps/details?id=com.lefu.insmart&hl=en_GB)) as an XLS file.
2. Open `index.html` in a web browser.
3. Click **Load file** and select your exported XLS file.

Your readings will load immediately. Use the range buttons (30d, 90d, 6m, 1y, All) to adjust the time window shown in the charts.

The app works entirely in the browser — no data is uploaded anywhere.

## Requesting support for other apps

Kilo currently supports exports from INSMART Health. To request support for another scale or health app, get in touch at [kilo@razorsharp.digital](mailto:kilo@razorsharp.digital).
